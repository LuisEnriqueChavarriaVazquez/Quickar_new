Dirección
  • Definición.

    El sistema de dirección se define como un conjunto de elementos que tienen la tarea de poder darle dirección a las ruedas delanteras para que este pueda tornar la dirección deseada por el conductor mediante un sistema mecánico.



  • Principios.

    El sistema de dirección está diseñado para facilitarle el trabajo al conductor, debido a que hoye en día los conductores no necesitan hacer uso de mucha fuerza para poder darle orientación a las llantas.


  • Tipos de dirección

    Existen una gran variedad de tipos de dirección, entre esos tipos de sistema de dirección podemos encontrar:

    Sistema de bolas recirculantes este tipo de dirección es más común encontrarlo en vehículos pesados y camiones.



    Sistema de piñón y cremallera es el más sencillo de todos los componentes dentro del sistema existe un lubricante para proteger el sistema.



    Sistema de dirección hidráulico este se acciona por medio de una bomba, y esta bomba a la vez es accionada por una correa que viene directamente del cigüeñal, por consecuente este sistema cuenta con un depósito.



    Sistema de dirección electrohidráulico este es casi igual al sistema hidráulico, pero a diferencia del anterior la fuerza o lo que acciona este sistema es el motor eléctrico del automóvil.



  • Elementos básicos.

    La dirección está constituida por varios elementos entre los cuales están:

    Brazo pitman


    Cajas de dirección


    Rótulas


    Columna de dirección


  • Servodirección.
    Este sistema consiste en sistema circula aceite por la venas de lubricación.

    Al accionar el volante la columna de dirección mueve solamente un distribuidor que por la acción de la bomba envía el aceite del cilindro que esta fijo al bastidor, dentro del cual un pistón se mueve en un sentido o en el otro según el lado en el que el volante está girando.

  • Cotas.
    Para que la conducción de un vehículo sea fiable y segura para un vehículo este ha de tener una dirección que reúna las siguientes condiciones:

    Debe ser semireversible

    No debe volver rápidamente ni ser irreversible.

    Este es posible con la dentadura de los engranes.

    Progresiva

    Significa que si damos al volante una vuelta completa, las ruedas deben dar unas dos vueltas y media con mucho menos esfuerzo de lo normal.


    Estabilidad

    La barra de dirección debe ser totalmente estable y debe tener una marcha recto con respecto del volante y del suelo lo cual se logra con las cotas de dirección. (Los tipos de cotas son de avance, salida, convergencia y divergencia)

    Avance

    Se toma en cuenta como la vertical del eje vertical del sentido longitudinal y la prolongación del pivote. El Angulo de esta inclinación suele ser de 2º

    Salida

    Se considera la vertical del con la prolongación del pivote en sentido totalmente transversal el cual suele ser de 5º


    Caída

    Se considera la horizontal de la mangueta y la misma mangueta en sentido transversal, igual suele ser de un Angulo de 2º

    Convergencia o divergencia

    Según el vehículo sea de tracción o propulsión de manera respectiva se considera la mangueta y la prolongación del eje (las ruedas no tienden a estar completamente paralelas en reposo) y tienen una inclinación normal de 2mm.
